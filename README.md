# Hackershare

Social bookmarks website for hackers. Alternative to delicious, hackernews, pocket.

[![CI](https://github.com/hackershare/hackershare/workflows/CI/badge.svg)](https://github.com/hackershare/hackershare/actions)

## Dependency

* Postgresql 12+
* Rails 6
* Stimulusjs 
* Turbolinks 5
* rails-ujs
